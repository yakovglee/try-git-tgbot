# try-git-tgbot

Учебный Telegram-бот на Python для отработки навыков работы с GitHub Issues.

## Что умеет бот

Бот выполняет простые математические операции над числами:

| Команда | Описание | Пример |
|---|---|---|
| `/pwr a,b` | Возводит `a` в степень `b` | `/pwr 2,5` → 32 |
| `/exp a` | Вычисляет экспоненту `e^a` | `/exp 1` → 2.718... |
| `/add a,b` | Сумма двух чисел | `/add 3,4` → 7 |
| `/sub a,b` | Разность двух чисел | `/sub 10,3` → 7 |
| `/mul a,b` | Произведение двух чисел | `/mul 6,7` → 42 |
| `/div a,b` | Частное двух чисел | `/div 10,2` → 5 |
| `/start`, `/help` | Справка по командам | — |

## Структура проекта

```
.
├── bot.py       # Точка входа, обработчики команд
├── logic.py     # Математическая логика
├── utils.py     # Парсинг аргументов из сообщений
├── config.py    # Загрузка конфигурации из .env
├── req.txt      # Зависимости
└── .env.example # Пример файла с переменными окружения
```

## Установка и запуск

1. Клонируй репозиторий:
   ```bash
   git clone https://github.com/yakovglee/try-git-tgbot.git
   cd try-git-tgbot
   ```

2. Установи зависимости:
   ```bash
   pip install -r req.txt
   ```

3. Создай файл `.env` на основе примера и укажи токен бота:
   ```bash
   cp .env.example .env
   ```
   Открой `.env` и заполни:
   ```
   TELEGRAM_BOT_TOKEN=your_token_here
   ```
   Токен можно получить у [@BotFather](https://t.me/BotFather).

4. Запусти бота:
   ```bash
   python bot.py
   ```
